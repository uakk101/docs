## Get started in four steps

<Steps>
  <Step title="Request API access">
    Sign up at the [CodeCrafting Hub dashboard](https://app.codecraftinghub.com/signup) and request API credentials for your organization.
  </Step>
  <Step title="Authenticate your requests">
    Add your API key to the `Authorization` header on every request. See [Authentication](/authentication) for details.
  </Step>
  <Step title="Register your first patient">
    Use `POST /patients` to create a patient record. Follow the [Patient Registration guide](/guides/patient-registration) for a full walkthrough.
  </Step>
  <Step title="Submit a claim">
    Create an encounter, attach diagnoses and procedures, verify eligibility, then submit via `POST /claims`. The [Claims Submission guide](/guides/claims-submission) walks you through each step.
  </Step>
</Steps>

<Note>
  All data exchanged with the CodeCrafting Hub API is encrypted in transit (TLS 1.2+) and at rest. The platform is fully HIPAA-compliant. See the [HIPAA & Compliance](/resources/hipaa) page for details on our BAA process.
</Note>
